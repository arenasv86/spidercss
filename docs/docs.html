Introduction

Spider CSS is a powerful SASS and Flexbox framework focused on performance and flexibility for medium and large projects. It isn't another UI Framework like Bootstrap, it just provides common functions, helpers, mixins and some basic UI elements for build webapps.

Thanks to SASS you can configure the project with your own variables and generate helpers and tools that make easier the UI development and improve your workflow.

Let the game begin!

Installation

There are four simple ways to start working with SpiderCSS:

Install it via Bower 
$ bower install spidercss --save
Install it via npm 
$ npm install spidercss --save
Clone the repo 
$ git clone https://github.com/arenasv86/spidercss.git
Download from GithubDownload

Usage

There are two ways to getting started with Spider CSS:

SASS (Recomended)

I highly recommend to work directly with the source files in the project development. You'll can modify all the variables and configure it at your taste for build a strong project structure.

  1. Copy the dist folder to your project assets folder
  2. Import spidercss.scss at the top of your .scss main file
Compiled CSS

If you want, you can use the compiled stylesheet in your project. Just find the file spidercss.min.css in the project dist folder and include it inside your document head.
